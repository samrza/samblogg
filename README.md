# samblogg - Personal Blog

A minimalist personal blog showcasing thoughts, writings, travel experiences, and achievements. Built with simplicity and freedom-first thinking in mind.

## 🌟 Features

- **Clean, Minimalist Design**: Focused on content with a distraction-free reading experience
- **Responsive Layout**: Optimized for all devices
- **Travel Timeline**: Visual journey through different locations with images and stories
- **Writings Section**: Collection of thoughts, book reviews, and recipes
- **Now Page**: Current activities, learning, and building projects
- **Questions**: Deep thoughts and philosophical musings
- **Achievements**: Milestones and personal growth journey

## 📁 Project Structure

```
samblogg/
├── pages/
│   ├── index.html          # Home page
│   ├── now.html           # Current activities
│   ├── travel.html        # Travel experiences
│   ├── writings.html      # Articles and thoughts
│   ├── questions.html     # Philosophical questions
│   └── achievements.html  # Personal milestones
├── images/                # Image assets
├── style.css             # Main stylesheet
└── README.md             # Project documentation
```
